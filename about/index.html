About

A curious mind
that ships.

Independent builder based between Zurich and Brussels, working at the intersection of policy, culture, and wellbeing.

I build small, opinionated web products end to end — idea, design, code, and the unglamorous part where it actually ships. The range is deliberate: a daily digest of EU foreign-policy news, a framework for designing a deliberate life, a live schedule for Zurich fitness studios, a Zen calendar of Japan's 72 microseasons.

What holds it together is a way of working: curiosity meets rigor. Wide-ranging inputs, exacting outputs. Everything I make is fast, dependency-light, and finished — usually static, always shipped.

What I work with

  • Plain HTML, CSS, and vanilla JavaScript — no frameworks unless they earn it
  • Python for automation, scraping, and data work
  • GitHub Actions for scheduled workers and self-updating sites
  • LLM tooling (Claude) woven into real products, not demos
