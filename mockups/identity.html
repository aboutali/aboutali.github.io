01 The idea

Data, made human.

Angelo builds small, opinionated web products across policy, culture, and wellbeing — work that is really about turning messy human things into something legible. The identity does the same to him: every project, discipline, and year becomes a mark. Inspired by Pentagram & Giorgia Lupi's data-driven identity for Capacity — not copied: the system, palette, and marks here are original.

03 The mark · generator

Data-portrait

The signature mark is a small matrix of dots, bars, and rings — a "data portrait." Each cell encodes an attribute (discipline, status, intensity); colour is drawn from the discipline palette. It is generative: one tool, endless on-brand marks, so every project or page can carry its own while staying unmistakably the system.

Press generate to see the system produce fresh, consistent marks — the way Capacity's identity ships with a pattern/chart generator rather than a fixed logo lock.

04 Pattern · generator

Data pattern

Zoom the mark logic out and it tiles into a pattern — a seating-chart-like field of marks for backgrounds, covers, and banners. Same rules, different scale.

Density, colour mix, and mark types stay within the system, so any generated field reads as the same brand.

05 Chart language

Branded data forms

Charts are part of the identity, not an afterthought — donut, bar, and dot-matrix, all styled in the palette. This is how Angelo presents his own work: the portfolio as a small dataset.

06 Colour

Warm ink, joyful accents

A warm ink-on-cream base (a vintage-print warmth, not stark white) carries a seven-hue accent set — one per discipline. Colour is structured, never decorative: a mark's hues always mean something.

07 Typography

One grotesque, two voices

A neo-grotesque does the heavy lifting (tight for display, plain for body); a monospace labels the data. Set in system fonts here so the site stays dependency-free — recommended licensed pairing: a geometric grotesque (e.g. a Futura/Mona-style display) with a humanist mono.
