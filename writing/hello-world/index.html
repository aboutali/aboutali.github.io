Hello, world — what this site is

2026-07-02 · 2 min read

This site is an index of things I've built and, from now on, an occasional place to write about building them. The projects range wider than a portfolio probably should — EU foreign-policy digests, a Zen calendar, a life-design framework — and that's the point. Curiosity meets rigor: follow the interesting thing, then finish it properly.

How it's built

Deliberately boring, in the best way. Plain HTML and CSS, no frameworks, no build step. A single GitHub Action runs daily: it pings each project to light a status dot, pulls the latest commits into the ticker on the homepage, and renders the guestbook straight from GitHub issues. The identity — Archivo, cobalt, warm paper — was designed first as a small brand system, then the site was built to follow it.

What to expect here

Short notes, mostly: decisions and trade-offs from the projects, occasional thoughts on policy and technology, things worth paying attention to. No schedule, no newsletter — when there's something worth writing, it shows up here.
